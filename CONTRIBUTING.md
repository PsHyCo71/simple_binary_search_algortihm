## How to Contribute

1. **Fork the repository** and create a new branch for your feature or fix:

    ```bash
    git checkout -b my-feature
    ```

2. Make your changes in a clear and organized manner. Please add comments where necessary.

3. Test your changes to ensure everything works as expected.

4. Commit your changes with a descriptive message:

    ```bash
    git commit -m "commit description"
    ```

5. Push your branch to your fork

    ```bash
    git push origin my-feature
    ```

6. Open a Pull Request describing your changes and why they should be merged.

## Suggestions and Ideas

Any proposal to make the program faster and more efficient are more than welcome. If you have ideas that require bigger changes, feel free to open an issue first so we can discuss it.

## Code of Conduct

Please be respectful and constructive.
